# Software Development Coursework

This repository contains programming projects and assignments completed as part of software development and computer programming coursework.

The goal of these projects was to develop foundational skills in application development, data management, and web technologies.

---

## Project Categories

### Python Projects
Scripts and small applications focused on data processing, automation, and program logic.

Topics include:
- file handling
- loops and control structures
- data processing
- scripting workflows

---

### C# Projects
Desktop applications and programming exercises built using C# and Visual Studio.

Topics include:
- object-oriented programming
- Windows Forms applications
- data structures
- user interface logic

---

### JavaScript Projects
Interactive web features built with JavaScript for dynamic website functionality.

Topics include:
- DOM manipulation
- event handling
- interactive UI elements
- client-side scripting

---

### HTML / CSS Projects
Website layouts and page structures built using core web technologies.

Topics include:
- responsive layouts
- semantic HTML
- styling with CSS
- user interface structure

---

### Database Projects
Database design and data management exercises.

Topics include:
- relational database design
- SQL queries
- data normalization
- database relationships

---

### Excel / Data Projects
Spreadsheet-based automation and data analysis projects.

Topics include:
- formulas and functions
- data filtering
- automation techniques
- data visualization
